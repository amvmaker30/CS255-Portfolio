# CS 255 Portfolio Reflection

## DriverPass Project Summary

The DriverPass project was about designing a new system for a company called DriverPass. The client wanted an online system that would help customers prepare for their driving test by scheduling driving lessons, taking online practice exams, and managing their accounts. The goal was to create a system that was easy to use for both customers and employees while keeping customer information secure.

## What I Did Well

I think I did a good job gathering the client's requirements and turning them into UML diagrams and a system design. The use case, activity, sequence, and class diagrams all worked together to show how the system would function. I also made sure the design included security features like user logins and different access levels.

## What I Would Revise

If I could go back and improve one part of the project, I would spend more time on the UML diagrams, especially the sequence and class diagrams. At first I found them confusing, but after practicing I understood how they connect to each other. I would make the relationships between the classes and the flow of the sequence diagram even clearer.

## Understanding the User's Needs

I used the interview transcript and business requirements to understand what DriverPass wanted from the system. I focused on making it simple for customers to schedule lessons, take practice tests, and manage their accounts. Keeping the user's needs in mind is important because a system that is difficult to use or doesn't solve the customer's problems won't be successful.

## My Approach to Software Design

When designing software, I like to start by understanding what the client needs before thinking about the technical details. I found that creating UML diagrams helped me organize my ideas and see how different parts of the system work together. In the future, I will continue using diagrams, gathering user requirements early, and reviewing my designs before development starts. This project showed me that planning first makes the development process much easier.
